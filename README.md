# Personal Finance Intelligence Agent

An AI-powered agent that analyzes personal financial data and produces tailored investment summaries and insights.

## Overview

This project is an autonomous agent that takes in personal financial data and generates clear, actionable investment summaries. The goal is to combine the analytical power of large language models with structured financial data to provide insights that are easy to understand and act on.

## Tech Stack

- **Python** — core language
- **LangChain** — framework for orchestrating the agent loop and tool calls
- **LLM API** — for natural language reasoning and summarization
- **Cursor** — development environment

## Features

- Ingests personal financial data
- Reasons over the data using an LLM-driven agent loop
- Produces concise investment summaries and insights tailored to the input

## Status

🚧 Work in progress. The core agent loop and summarization pipeline are functional; ongoing work focuses on improving reasoning quality and expanding the agent's tool set.
